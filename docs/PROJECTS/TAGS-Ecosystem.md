# TAGS Ecosystem

**Organization:** [The Angry Gamer Show Productions](https://github.com/theangrygamershowproductions)

## Overview

The current TAGS build phase brings together software, automation,
documentation, and infrastructure projects operated under The Angry Gamer Show
Productions, LLC.

## Operating Model

Chad Reesey is the founder and CEO of The Angry Gamer Show Productions, LLC, a
United States Air Force veteran-owned and operated Florida company.

The current build phase is operated as an **AI-assisted solo human operator**
model. This describes the human operating model for the current phase and does
not imply sole authorship of every historical commit, dependency, generated
output, or third-party component.

## Current and Recent Project-Stack Exposure

Current or recent TAGS project work includes TypeScript and Fastify; React,
Vite, Tailwind CSS, and shadcn/ui; Traefik and TrueNAS SCALE; and MCP SDK
integration where applicable.

This is project-stack exposure. It does not mean every technology is used by
every project, and supporting technologies are not presented as separate
core-capability badges.

## Delivered Work, Plans, and Experiments

A component is described as delivered only when current public source or an
owner-confirmed public statement supports that status. Roadmaps, possible
integrations, and future services must remain labeled as plans or experiments.

This overview does not publish private topology, private-source metrics,
internal project counts, or non-public implementation details.

## Scope Boundary

The TAGS project descriptions do not claim:

- A universal architecture shared by every project
- Organization-wide coverage or quality percentages
- A single project controlling or defining every other project
- Readiness, performance, security, or business outcomes without a
  reproducible public basis
