# DevOnboarder

**Repository:** [theangrygamershowproductions/DevOnboarder](https://github.com/theangrygamershowproductions/DevOnboarder)

## Overview

DevOnboarder is a multi-service onboarding-automation project with Discord,
web, and CI/CD components.

## Public Source Scope

The public repository contains work in these areas:

- Python services and automation scripts
- Discord-based onboarding and notification components
- Web and API components
- GitHub Actions workflows and CI checks
- Documentation validation and setup guidance
- Container and development-environment support

Individual checks, services, and workflows have their own repository-defined
requirements. This overview does not assign one universal metric or readiness
label to the project.

## Relationship to TAGS

DevOnboarder is one project associated with work under The Angry Gamer Show
Productions. Each TAGS project retains its own architecture and source of
authority; this overview does not assign organization-wide responsibilities
to DevOnboarder.

## Delivered Work and Plans

This page describes components visible in the current public source. Proposed
integrations, deployment environments, monitoring work, and other roadmap
items are plans unless and until their implementation is visible and
documented.
