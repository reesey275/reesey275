# Codex Agents Overview

This directory contains prompt files for the Codex agents that automate updates across the repository. The table below lists each agent along with its role and scope for quick reference.

| Agent File | Role | Scope | Description |
|-----------|------|-------|-------------|
| [agent_certs_sync.md](agent_certs_sync.md) | maintainer | credentials | Synchronizes certificates and credentials with external services. |
| [agent_pages_deployer.md](agent_pages_deployer.md) | maintainer | website | Builds the static site and deploys it to GitHub Pages. |
| [agent_readme_composer.md](agent_readme_composer.md) | writer | documentation | Composes README files from project information. |
| [agent_profile_writer.md](agent_profile_writer.md) | writer | portfolio | Creates or updates project summaries across `/projects/*/README.md`. |
